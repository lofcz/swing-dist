# swing-dist

This is ESM version of `https://github.com/Brugarolas/swing`.

## Build

1. `npm install`
2. `npm run build`
3. use `swing.js` bundle, swing is exposed as `window.swingLib` (alternatively, prebuilt `swing.js` file is provided in the repository)

## Edit source code

1. `cd src/src`
2. do your changes
3. rebuild
